Описание возможностей приложения:

  • У вас есть возможность создать собственный аккаунт
  • Так же вы имеете свою собственную корзину, куда можете складывать покупки
  • У вас есть вожность добовлять товары в избранное
  • Для создания приложения использовались библиотеки: Angular Material, ng-boorstap
  • Использовались pipe's, interseptor's и многое другое.
  • Реализована пагинация на 10 элементов
  • Есть возможность сортировки товаров по цене / стоимости
  • У элементов у который высокий рейтинг = 5, отображается звёздочка :)
  • В приложении есть всё что надо :)
